## PART 01 伤害乘区

### 核心伤害公式

```
常规伤害 = 基础伤害区 × 增伤区 × 暴击区 × 防御区 × 抗性区 × 减易伤区 × 失衡易伤区 × 特殊乘区

贯穿伤害 = 基础伤害区 × 增伤区 × 暴击区 × 贯穿增伤区 × 抗性区 × 减易伤区 × 失衡易伤区 × 特殊乘区
```

> 游戏伤害显示向上取整，多段伤害每段取整后相加。

---

### 1.1 基础伤害区

```
基础伤害区 = Σ(伤害倍率 × 对应属性)
```

#### 属性计算

```
最终属性 = 初始属性 × (1 + 最终属性加成/降低) + 最终属性固定值加成
初始属性 = 基础属性 × (1 + 初始属性加成/降低) + 初始属性固定值加成
```

**适用属性**: 攻击力、生命值、防御力、冲击力、贯穿力、异常精通、异常掌控、能量自动回复、闪能自动累积

#### 贯穿伤害

贯穿伤害基于**贯穿力**结算，本质是特殊的攻击力。

**仪玄贯穿力**:
```
仪玄贯穿力 = 攻击力 × 0.3 + 生命值上限 × 0.1
```

> 转化比例每个命破代理人单独设置

#### 真实伤害

直接降低目标生命值，仅计算基础伤害区的纯伤害。

| 异常状态 | 伤害倍率公式 |
|---------|------------|
| 火属性 [灼烧] | `450% + floor(T / 0.5) × 50%` |
| 电属性 [感电] | `450% + floor(T) × 125%` |
| 以太属性 [侵蚀] | `450% + floor(T / 0.5) × 62.5%` |
| 冰属性 [霜寒] | `450% + floor(T) × 7.5%` |
| 物理 [畏缩] | `450% + floor(T) × 7.5%` |
| 玄墨属性 [侵蚀] | `450% + floor(T / 0.5) × 62.5%` |
| 烈霜属性 [霜寒] | `600% + floor(T) × 75%` |

T = 异常状态剩余持续时间，floor() = 向下取整

---

### 1.2 增伤区

```
增伤区 = 1 + Σ增伤
```

**有效范围**: `[0, 6]`

- 包含所有"造成的伤害提升/降低"效果，默认加算
- 属性: 火、电、冰、物理、以太（烈霜→冰，玄墨→以太）
- 贯穿增伤和异常增伤不属于该乘区
- "造成的伤害降低"属于增伤区（负值）

---

### 1.3 暴击区

```
暴击区 = 1 + 暴击伤害  (暴击时)
暴击区 = 1              (未暴击时)
```

**暴击期望**: `暴击期望 = 1 + 暴击率 × 暴击伤害`

| 角色类型 | 基础暴击率 | 基础暴击伤害 |
|---------|----------|------------|
| 代理人 | 5% | 50% |
| 敌人 | 0% | 50% |

**有效范围**: 暴击率 `[0, 1]`，暴击伤害 `[0, 5]`，暴击区 `[1, 6]`

---

### 1.4 防御区

```
防御区 = 攻击方等级基数 / (受击方有效防御 + 攻击方等级基数)

受击方有效防御 = 受击方防御 × (1 - 攻击方穿透率%) - 攻击方穿透值  ≥ 0

受击方防御 = 基础防御 × (1 + 防御加成% - 防御降低% - 无视防御%) × (1 - 穿透率%) - 穿透值
```

> 无视防御和减防**加算**，穿透率和减防**乘算**

#### 等级基数

| 等级 | 等级基数 |
|-----|--------|
| 1-60 | 50-794 |
| >60 | 794 |

#### 敌人基础防御

```
某等级基础防御 = 1级基础防御 / 50 × 该等级攻击方等级基数
```

| 敌人类型 | 1级基础防御 |
|---------|----------|
| 普通敌人 | 36-46 |
| 精英敌人 | 45-54 |
| 杜拉罕 | 58 |
| 首领敌人 | 60 |

> 敌人基础防御60级达上限952.8，之后不再成长

#### 贯穿伤害

贯穿伤害**跳过防御区**，对首领敌人伤害为常规伤害的**2.2倍**

---

### 1.5 抗性区

```
抗性区 = 1 - 受击方抗性 + 受击方抗性降低 + 攻击方无视抗性
```

**有效范围**: `[0, 2]`

| 属性类型 | 抗性 |
|---------|-----|
| 弱点属性 | -20% |
| 抗性属性 | +20% |

> 特例: 庞培电抗性40%；离子体·多佩冈亚·暗渊惩戒者、秽息司祭冰抗性40%

**视觉效果**: 抗性 ≤ -20% → 闪白；抗性 ≥ 20% → 灰色

---

### 1.6 减易伤区

```
减易伤区 = 1 + 受击方易伤 - 受击方减伤
```

**有效范围**: `[0.2, 2]`（无法无伤）

---

### 1.7 失衡易伤区

```
失衡易伤区 = 1 + 失衡易伤倍率  (失衡时)
失衡易伤区 = 1 + 未失衡时失衡易伤倍率  (未失衡时)
```

| 状态 | 范围 |
|-----|-----|
| 失衡时 | [0.2, 5] |
| 未失衡时 | [1, 3] |

> 代理人和邦布不具有失衡条，不计入此乘区

**默认失衡易伤倍率**: 25%（特殊机制）、50%（大部分）、100%

---

### 1.8 贯穿增伤区

```
贯穿增伤区 = 1 + 贯穿增伤
```

**有效范围**: `[0.2, 9]`

命破代理人贯穿增伤数值显示在属性面板上。

---

### 1.9 特殊乘区（距离衰减）

| 类型 | 衰减机制 |
|-----|---------|
| 默认型（比利、丽娜、朱鸢） | 超过15m时伤害衰减25%，每继续远离5m继续衰减25% |
| 格莉丝型 | 超过15m时伤害衰减30% |

## PART 02 失衡

### 2.1 失衡值

```
失衡值累积 = Σ基础失衡区 × 失衡抗性区 × 失衡值提升区 × 受到失衡值提升区 × 距离衰减区
```

#### 2.1.1 失衡比例

```
失衡比例 = 失衡值累积 / 失衡值上限 × 100%
```

以整数形式显示在怪物失衡条右侧，向下取整。

#### 失衡比例锁定

- 具有[秽盾]: 99.5%
- 秽息蚀者·阿瓦鲁斯: 99%
- 提尔锋、阿佩卡被寄生: 1%

#### 失衡值上限

同名敌人分为强化型和非强化型：
- 普通敌人: 强化型 = 非强化型 × 1.5
- 精英敌人: 强化型 = 非强化型 × 1.13-1.15

#### 2.1.2 基础失衡区

```
基础失衡区 = 冲击力 × 失衡倍率
```

**有效范围**: 冲击力 `[0, 1000]`

#### 2.1.3 失衡抗性区

```
失衡抗性区 = 1 - 受击方失衡抗性
```

**有效范围**: `[0, 2]`。弱点属性-20%，抗性属性+20%

#### 2.1.4 失衡值提升区

```
失衡值提升区 = 1 + 造成失衡值提升 - 造成失衡值降低
```

**有效范围**: `[0, 4]`

#### 2.1.5 受到失衡值提升区

```
受到失衡值提升区 = 1 + 受到失衡值提升 - 受到失衡值降低
```

**有效范围**: `[0, 4]`

#### 2.1.6 直接失衡值累积/回复

**累积**: [部位破坏]、塔纳托斯受招架支援额外25%失衡值上限、阿劳恩30%

**回复**: [惊骇阴影]回复3%-5%失衡值

#### 2.1.7 失衡值自动回复

| 敌人类型 | 自动回复 |
|---------|---------|
| 匪祸侵蚀体·贪婪射手、游魂等 | 5%/秒 |
| 突击炮手、重装炮手等 | 1%/秒 |
| 侵蚀[活化效应] 普通 | 2%/秒 |
| 侵蚀[活化效应] 精英 | 1%/秒 |
| 侵蚀[活化效应] 首领 | 0.5%/秒 |

---

### 2.2 失衡状态

1. 累积失衡比例到100%
2. 失衡条红黄闪烁（不具有失衡易伤），敌人可自由行动
3. 再次受到可触发失衡状态的攻击时进入失衡状态

> [招架支援]不具有可触发失衡状态的标签

#### 连携机会

- 首领: 3次
- 精英: 2次
- 普通: 1次

连携补偿时间默认只叠加1层，不刷新。

---

### 2.3 失衡恢复

#### 失衡恢复速度

| 敌人类型 | 初始恢复速度 | 恢复时间 |
|---------|------------|---------|
| 袭扰猎兵等 | 50%/秒 | 2秒 |
| 其他普通敌人 | 15.38%/秒 | 6.5秒 |
| 阿劳恩、冥宁芙等 | 14.28%/秒 | 7秒 |
| 首领敌人 | 8.33%/秒 | 12秒 |
| '卫士'、法布提等 | 7.69%/秒 | 13秒 |
| 其他精英敌人 | 10%/秒 | 10秒 |

#### 恢复时间调整

**百分比调整**: 通过调整失衡恢复速度实现

**固定延长**: 使失衡恢复延迟固定时间后开始（与恢复速度无关）

## PART 03 属性异常

### 3.1 异常效果

#### 冻结和霜寒

- 施加[冻结]同时施加[霜寒]
- [冻结]默认持续3.5秒
- 冰[霜寒]持续10秒，烈霜[霜寒]持续20秒
- [霜寒]期间暴击伤害提升10%

**冻结时长**: `冻结时长 = 2 + 1.5 × (1 - 冻结时间抵抗)`

| 敌人类型 | 冻结时间抵抗 | 持续时间 |
|---------|------------|---------|
| 普通 | 0% | 3.5秒 |
| 精英 | 50% | 2.7秒 |
| 首领 | 75% | 2.4秒 |

**碎冰**: 500%攻击力冰/烈霜异常伤害

#### 强击和畏缩

- 强击: 713%攻击力物理异常伤害，4级打断等级
- 畏缩: 持续10秒，期间受到失衡值提升7.5%

#### 侵蚀

- 持续10秒，攻击造成62.5%攻击力以太/玄墨异常伤害，每0.5秒1次
- 仪玄触发时伤害为玄墨属性

#### 感电

- 持续10秒，攻击造成125%攻击力电异常伤害，每秒1次至多16次

#### 灼烧

- 持续10秒，每0.5秒造成50%攻击力火异常伤害

---

### 3.2 异常积蓄

```
异常积蓄值 = 基础异常积蓄值 × 异常掌控区 × 异常积蓄效率区 × 异常积蓄抗性区 × 距离衰减区
```

#### 异常掌控区

```
异常掌控区 = 异常掌控 / 100
```

**有效范围**: `[0, 3]`（计算时异常掌控向下取整）

#### 异常积蓄效率区

```
异常积蓄效率区 = 1 + 异常积蓄效率提升 + 异常积蓄值提升 + 异常积蓄效率降低
```

#### 异常积蓄抗性区

```
异常积蓄抗性区 = 1 - 异常积蓄抗性 + 异常积蓄抗性降低
```

| 敌人类型 | 弱点 | 抗性 |
|---------|------|------|
| 普通 | -0.2 | +0.2 |
| 曼德拉冰、以太 | +2.8 | - |
| 曼德拉物理、火、电 | +3.0 | - |

#### 触发阈值

| 触发次数 | 普通 | 精英 | 首领 |
|---------|------|------|------|
| 物理异常 | 4.8 | 18 | 24 |
| 其他异常 | 4 | 15 | 20 |

> 物理异常的阈值是其他属性的1.2倍；同种属性异常3秒冷却

---

### 3.3 异常伤害

```
异常伤害 = 基础伤害区 × 增伤区 × 异常精通区 × 防御区 × 抗性区 × 减易伤区 × 失衡易伤区 × 伤害等级区 × 异常增伤区 × 异常暴击区
```

> 不包含常规暴击区，基础伤害区默认基于攻击力

#### 异常精通区

```
异常精通区 = 异常精通 / 100
```

**有效范围**: `[0, 10]`

#### 伤害等级区

```
伤害等级区 = trunc(1 + 1/59 × (等级 - 1), 4)
```

| 等级 | 伤害等级区 |
|-----|----------|
| 1 | 1 |
| 60 | 2 |

#### 异常增伤区

```
异常增伤区 = 1 + 异常伤害提升
```

**有效范围**: `[0, 3]`

大部分效果不作用于[紊乱]伤害。

#### 异常暴击区

```
异常暴击区 = 1 + 异常暴击伤害  (暴击时)
异常暴击区 = 1                  (未暴击时)
```

**有效范围**: `[1, 3]`

#### 虚拟代理人机制

记录造成异常积蓄时代理人的属性，基于当次攻击的异常积蓄占比加权平均。

---

### 3.4 紊乱

一种属性异常持续期间触发其他类型异常时，新异常覆盖原状态并结算[紊乱]。

#### 紊乱伤害

| 异常状态 | 伤害倍率 |
|---------|---------|
| 火属性 [灼烧] | `450% + floor(T / 0.5) × 50%` |
| 电属性 [感电] | `450% + floor(T) × 125%` |
| 以太属性 [侵蚀] | `450% + floor(T / 0.5) × 62.5%` |
| 冰属性 [霜寒] | `450% + floor(T) × 7.5%` |
| 物理 [畏缩] | `450% + floor(T) × 7.5%` |
| 玄墨属性 [侵蚀] | `450% + floor(T / 0.5) × 62.5%` |
| 烈霜属性 [霜寒] | `600% + floor(T) × 75%` |

#### 紊乱失衡

```
[紊乱]失衡值累积 = 基础失衡区 × 失衡抗性区 × 失衡值提升区 × 受到失衡值提升区 × 失衡等级区

失衡等级区 = 1 + 0.0075 × 等级
```

| 等级 | 失衡等级区 |
|-----|----------|
| 1 | 1.0075 |
| 60 | 1.45 |

> 紊乱失衡倍率默认200%

## PART 04 能量系统

### 4.1 能量

```
能量回复值 = (Σ基础能量回复值 + 能量自动回复 × 时间) × 能量获得效率区

能量获得效率区 = 1 + 能量获得效率
```

**有效范围**: `[0, 3]`

#### 能量自动回复状态

| 状态 | 触发条件 | 持续时间 |
|-----|---------|---------|
| 短接战 | 受制或极限闪避 | 3秒 |
| 长接战 | 招架支援 | 7秒 |
| 长接战 | 攻击间隔0.5-5秒 | 7秒 |

#### 能量球

| 敌人类型 | 能量球 |
|---------|-------|
| 普通 | 0 |
| 精英 | 3 |
| 首领 | 5 |

每个能量球回复1%，前后台都能受益。

---

### 4.2 闪能

```
闪能累积值 = (Σ基础闪能累积值 + 闪能自动累积 × 时间) × 闪能获得效率区

闪能获得效率区 = 1 + 闪能获得效率
```

**有效范围**: `[0, 3]`

**仪玄初始**: 闪能上限120，闪能自动回复2点/秒

---

### 4.3 喧响值

1.4版本后各代理人喧响槽**相互独立**，初始上限3000。

#### 喧响等级

| 喧响值 | 等级 |
|-------|-----|
| 1000 | [喧] |
| 2000 | [特] |
| 3000 | [极] |

> 常规终结技需[极]等级消耗3000点

#### 喧响值回复

```
喧响值回复 = 基础喧响值回复 × 喧响获得效率区 × 伴随获得效率
```

#### 喧响获得效率区

```
喧响获得效率区 = 1 + 喧响值获得效率
```

**有效范围**: `[0, 3]`

#### 喧响伴随获得效率

| 代理人 | 伴随获得效率 |
|-------|------------|
| 绝大部分 | 50% |
| 可琳、比利、悠真、柏妮思、朱鸢 | 52.5% |

## PART 05 秽息

### 5.1 秽浊流界与秽盾

#### 秽浊流界效果

- 防御力提高**80%**
- 受到的伤害降低**25%**（秽蚀·色雷斯人60%）
- [秽息浸染]值提高**30%**
- 抗打断能力提升2级

> 连携技、终结技、招架支援命中不受影响

#### 秽盾

**效果**:
- 不会进入失衡状态
- 失衡百分比上限锁定至99.5%
- 不提供属性异常抗性
- 无视施加异常状态时的打断
- 不会进入[冻结]状态

**参数**:

| 参数 | 默认 | 高档位 |
|-----|------|-------|
| 上限 | 2200 | 3500 |
| 衰减速度 | 1%/秒 | - |
| 衰减下限 | 10% | - |

> 秽蚀·色雷斯人特化秽盾: 上限4500，不衰减

**高档位秽盾敌人**: 强化秽息蚀者·阿瓦鲁斯、秽息司祭、秽蚀·色雷斯人、秽蚀·多佩冈亚·简、秽蚀·多佩冈亚·暗渊惩戒者

---

### 5.2 秽盾净除

打破[秽盾]时:
- 受到[秽盾净除]伤害（默认15%最大生命值）
- 队伍回复50%能量或闪能
- 清除[秽浊流界]

**版本变化**: 2.2前15% → 2.2后2.5%

#### 秽盾削减

```
秽盾削减值 = 基础秽盾削减值 × 秽盾削减效率区 × 秽盾被削减效率区

秽盾削减效率区 = 1 + 秽盾削减效率
秽盾被削减效率区 = 1 + 秽盾被削减效率
```

**有效范围**: `[0.2, 3]`

> 邦布攻击不具有秽盾削减能力
