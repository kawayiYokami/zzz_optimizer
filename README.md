# 绝区零优化器（UpDown）

面向《绝区零》的本地配装/伤害/优化工具：导入角色与装备数据后，提供伤害期望计算、Buff/环境配置与驱动盘组合搜索。

本项目以“本地优先”为原则：数据与存档默认保存在浏览器本地，不依赖后端服务。

## 功能

- 数据管理：导入/导出角色、音擎、驱动盘等存档数据
- 伤害计算：按乘区口径计算直伤/异常/紊乱等期望
- 装备优化：多线程 Worker 搜索驱动盘组合，输出 TopN 方案
- 队伍与环境：前台/后台角色管理，Buff 开关，敌人/失衡/秽盾等环境参数
- 多技能口径：支持选择多个技能并按次数叠加（rotation 权重）

## 开发与运行

```bash
cd web/optimizer
pnpm install
pnpm dev
```

启动后访问 `http://localhost:8522`。

## 测试与构建

```bash
cd web/optimizer
pnpm test -- --run
pnpm build
```

## 目录结构（节选）

- `web/optimizer/`：Web 前端与优化器核心
- `web/optimizer/src/optimizer/`：优化器、预计算、Worker
- `web/optimizer/public/game-data/`：静态游戏资料（如图标/索引）
